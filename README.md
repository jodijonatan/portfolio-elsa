# Elsa's Portfolio

A personal portfolio website showcasing the creative works of Elsa, a talented novelist. This site highlights her novels, music, and professional profile, built with modern web technologies for an engaging user experience.

## Features

- **Home**: An inviting introduction to Elsa's world as a novelist.
- **About**: Detailed profile, skills in creative tools like Picsart, Ibis Paint, Pinterest, Artflow, and CapCut.
- **Works (Karya)**: Showcase of Elsa's literary creations and projects.
- **Music**: A collection of music tracks with album covers.
- **Contact**: Easy way to get in touch for collaborations or inquiries.

## Tech Stack

- **Frontend**: React 19.1.1
- **Build Tool**: Vite 7.1.7
- **Styling**: Tailwind CSS 4.1.13
- **Animations**: Framer Motion 12.23.22
- **Icons**: Lucide React 0.544.0
- **Linting**: ESLint 9.36.0

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/jodijonatan/elsa-portfolio.git
   cd elsa-portfolio
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Usage

- Navigate through the sections using the navbar.
- Explore Elsa's novels in the Works section.
- Listen to music tracks in the Music section.
- Use the Contact form to reach out.

## Scripts

- `npm run dev`: Start the development server.
- `npm run build`: Build the project for production.
- `npm run lint`: Run ESLint for code quality checks.
- `npm run preview`: Preview the production build locally.
